# MovieLens Analysis — Техническое задание

## Общие инструкции

- В глобальной области **запрещён код** — используйте только функции.
- Любое **необработанное исключение** делает работу недействительной.
- Используемый интерпретатор: **Python 3**.
- Разрешённые библиотеки:
  - `os`, `sys`, `urllib`, `requests`, `beautifulsoup`, `json`
  - `pytest`, `collections`, `functools`, `datetime`, `re`
- Итоговый отчёт необходимо создать в **Jupyter Notebook**.
- В ноутбуке запрещены дополнительные импорты — используем только модуль.
- Если вам нужны вспомогательные функции — заранее добавьте их в модуль.

---

## Глава IV — Обязательная часть

Вы должны подготовить два файла:

1. `movielens_analysis.py` — модуль с классами и методами  
2. `movielens_report.ipynb` — аналитический отчёт (использует только ваш модуль)

### Данные

- Используйте уменьшенную версию **MovieLens**.
- Внимательно прочитайте `README.txt`.
- Загружайте **первые 1000 строк** каждого файла.

---

## Модуль (`movielens_analysis.py`)

Необходимо реализовать **5 классов**:

- `Ratings`
- `Tags`
- `Movies`
- `Links`
- `Tests` (класс с тестами)

Каждый рабочий класс должен:

- загружать соответствующий файл MovieLens,
- обрабатывать только первые 1000 записей,
- предоставлять методы анализа и обработки данных (как минимум те, что указаны в шаблоне проекта).

### Класс `Tests`

Создайте тесты на **pytest** для каждого метода всех классов:

Тесты должны проверять:

- корректность возвращаемых типов,
- корректность типов элементов списков,
- правильность сортировки возвращаемых данных (если метод сортирует),
- корректность поведения методов на валидных входах.

Запустите все тесты до начала работы над отчётом.

## Отчёт (`movielens_report.ipynb`)

Требования:

- Использовать **только методы вашего модуля**.
- Каждый кодовый блок **обязательно** должен содержать магическую команду:
`%timeit`

- Никаких дополнительных импортов.
- Структура отчёта — произвольная, но должна представлять **аналитическую историю**
о данных MovieLens.
- Можно и нужно включать текстовые пояснения.
