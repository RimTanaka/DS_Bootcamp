# Python Basics — Техническое Задание

Набор упражнений по основам Python: типы данных, работа с файлами, словарями, множествами, строками и аргументами командной строки.

Все упражнения должны быть выполнены:

- в отдельных папках `ex00/`, `ex01/`, ..., `ex09/`
- с точными именами файлов
- без лишних импортов (если не указано)
- поведение программ должно строго соответствовать примерам

---

## Exercise 00 — Типы данных

**Папка:** `ex00/`  
**Файл:** `data_types.py`  
**Разрешённые импорты:** никаких

Задача:

- создать функцию `data_types()`
- объявить 8 переменных разных типов
- вывести список типов, например:

`[int, str, float, bool, list, dict, tuple, set]`


- типы должны определяться динамически, без ручного вписывания
- в конце скрипта обязательно:

```python
if __name__ == '__main__':
    data_types()
```

  - тикер?

- правила:
  - регистр игнорируется,
  - пробелы игнорируются,


## Exercise 07 — Множества в маркетинге

**Папка:** `ex07/`  
**Файл:** `marketing.py`  
**Разрешённый импорт:** `sys`

**Даны списки:** `clients`, `participants`, `recipients`.

**Требуется получить:**

- клиентов, которые **не видели письмо** → `call_center`;
- участников, которые **не являются клиентами** → `potential_clients`;
- клиентов, которые **не участвовали в событии** → `loyalty_program`.

**Требования:**

- преобразовывать списки в множества;
- использовать операции множеств;
- скрипт принимает аргумент: `call_center`, `potential_clients`, `loyalty_program`;
- неверное имя → исключение.

## Exercise 08 — Работа со строками как со списками

### Часть 1 — `names_extractor.py`

- аргумент: путь к файлу с e-mail вида `name.surname@corp.com`;
- строки разделены `\n`;
- выходной файл: `employees.tsv`;
- формат колонок: `Name\tSurname\tE-mail`;

- аргумент: e-mail;

- запрещено использовать `.format()`;
