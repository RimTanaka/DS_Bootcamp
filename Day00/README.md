# Shell Rush — Техническое задание

Набор упражнений по работе с API HeadHunter, JSON, CSV и обработкой данных с помощью shell-скриптов.

Все скрипты должны быть:
- исполняемыми,
- написанными под интерпретатор **/bin/sh**,
- размещены в соответствующих директориях `ex00/`, `ex01/`, …  
- выводы (JSON/CSV) также должны быть помещены в соответствующие папки.

---

## Exercise 00 — Первый shell-скрипт

**Папка:** `ex00/`  
**Файлы:** `hh.sh`, `hh.json`  
**Разрешённые функции:** `curl`, `jq`

Задача:

- скрипт принимает название вакансии (например, `"data scientist"`);
- делает запрос к HeadHunter API;
- загружает **первые 20 вакансий** по поисковому запросу;
- сохраняет результат в файл **hh.json**;
- JSON должен быть отформатирован построчно (каждое поле — на новой строке).

---

## Exercise 01 — Преобразование JSON → CSV

**Папка:** `ex01/`  
**Файлы:** `filter.jq`, `json_to_csv.sh`, `hh.csv`  
**Разрешённые функции:** `jq`

Задача:

- скрипт `json_to_csv.sh` запускает `jq` с фильтром из `filter.jq`;
- в CSV должны попасть поля:
  - `id`
  - `created_at`
  - `name`
  - `has_test`
  - `alternate_url`
- результат сохранить в **hh.csv** с **заголовками** (header).

---

## Exercise 02 — Сортировка CSV

**Папка:** `ex02/`  
**Файлы:** `sorter.sh`, `hh_sorted.csv`  
**Разрешённые функции:** `cat`, `sort`, `head`, `tail`

Задача:

- отсортировать файл **hh.csv** (из ex01) по:
  1. `created_at`  
  2. затем по `id`  
  в **возрастающем** порядке;
- сохранить результат в **hh_sorted.csv**;
- CSV должен сохранить заголовок первой строки.

---

## Exercise 03 — Очистка текстовых данных

**Папка:** `ex03/`  
**Файлы:** `cleaner.sh`, `hh_positions.csv`  
**Разрешённые функции:** без ограничений

Задача:

- очистить поле `name`, оставив:
  - только `Junior`, `Middle`, `Senior`;
  - если в названии несколько — оставить все (например, `Middle/Senior`);
  - если ни одного нет — использовать `-`;
- сохранить результат в **hh_positions.csv**;
- CSV должен сохранять заголовок и сортировку из предыдущего упражнения.

---

## Exercise 04 — Описательная статистика

**Папка:** `ex04/`  
**Файлы:** `counter.sh`, `hh_uniq_positions.csv`  
**Разрешённые функции:** без ограничений

Задача:

- подсчитать количество уникальных значений в колонке `name` из файла ex03;
- отсортировать результат по количеству в **убывающем** порядке;
- сохранить как CSV с колонками:

`"name","count"`

- результат — файл **hh_uniq_positions.csv**.

---

## Exercise 05 — Партиционирование и объединение

**Папка:** `ex05/`
**Файлы:** `partitioner.sh`, `concatenator.sh`
**Разрешённые функции:** без ограничений

### partitioner.sh:

- разбить файл из Exercise 03 на набор CSV по дате `created_at`;
- названия файлов — даты;
- каждый файл должен содержать заголовки.

### concatenator.sh:

- собрать все файлы, созданные partitioner.sh, обратно в один CSV;
- итоговый CSV должен быть идентичен файлу из Exercise 03.

---

Все скрипты должны быть исполняемыми и находиться в своих директориях.
