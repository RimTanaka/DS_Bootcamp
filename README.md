# Этот репозиторий содержит набор задач и проектов, охватывающих путь обучения от работы с Unix-утилитами и API до продвинутой аналитики данных, визуализации и машинного обучения.


## **Day00 — Shell Rush**
Работа с API, JSON, CSV, сортировкой, фильтрацией и партиционированием данных через shell-скрипты.

## **Day01 — Python Basics**
Основы Python: типы данных, работа с файлами, списками, словарями, строками, множества, Caesar cipher.

## **Day02 — OOP Python Rush**
Создание собственных классов, методов, конструкторов, вложенных классов, наследования, отчётов и логирования.

## **Day03 — Virtual Environment & Libraries**
Создание виртуальных окружений, управление зависимостями, BeautifulSoup, профилирование, тестирование.

## **Day04 — Benchmarking**
Измерение производительности: цикл vs list comprehension, map/filter/reduce, Counter, генераторы и память.

## **Day05 — Pandas**
Предобработка данных, агрегации, очистка, оптимизации, join, pivot table, работа с большими датасетами.

## **Day06 — SQL + Pandas**
SELECT, JOIN, подзапросы, витрины данных, агрегации, корреляции и A/B-тестирование.

## **Day07 — Data Visualization**
Matplotlib, Seaborn, Plotly, heatmap, scatter matrix, анимации, комбинированная аналитика SQL + Pandas.

## **Day08 — Machine Learning**
Binary classification, multiclass, decision boundaries, overfitting, regression и clustering.

## **Team00 — MovieLens Analysis**
Модульная аналитика датасета MovieLens: классы для работы с рейтингами, тегами, фильмами и ссылками, unit-тесты на pytest и финальный отчёт в Jupyter Notebook на основе собственного модуля.
